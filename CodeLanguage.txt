General-Purpose Programming Languages

Python

What it is: A high-level, versatile language known for its simplicity and readability.
Uses: Web development, data analysis, machine learning, scripting, automation.
Example:
python
Copy code
print("Hello, World!")

JavaScript

What it is: A core web development language used to make web pages interactive.
Uses: Frontend (with HTML/CSS), backend (Node.js), web applications.
Example:
javascript
Copy code
console.log("Hello, World!");

Java

What it is: A robust, object-oriented language with a "write once, run anywhere" philosophy.
Uses: Enterprise software, Android apps, backend development.
Example:
java
Copy code
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

C++

What it is: An extension of C with object-oriented features, widely used for system and performance-critical applications.
Uses: Game development, operating systems, high-performance applications.
Example:
cpp
Copy code
#include <iostream>
int main() {
    std::cout << "Hello, World!";
    return 0;
}

HTML

What it is: The standard language for structuring web pages.
Uses: Building the skeleton of websites.
Example:
html
Copy code
<h1>Hello, World!</h1>

CSS

What it is: Used for styling HTML content.
Uses: Designing visually appealing web pages.
Example:
css
Copy code
h1 {
    color: blue;
}

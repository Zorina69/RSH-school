General-Purpose Programming Languages

Python

What it is: A high-level, versatile language known for its simplicity and readability.
Uses: Web development, data analysis, machine learning, scripting, automation.
Example:
python
Copy code
print("Hello, World!")

JavaScript

What it is: A core web development language used to make web pages interactive.
Uses: Frontend (with HTML/CSS), backend (Node.js), web applications.
Example:
javascript
Copy code
console.log("Hello, World!");

Java

What it is: A robust, object-oriented language with a "write once, run anywhere" philosophy.
Uses: Enterprise software, Android apps, backend development.
Example:
java
Copy code
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

C++

What it is: An extension of C with object-oriented features, widely used for system and performance-critical applications.
Uses: Game development, operating systems, high-performance applications.
Example:
cpp
Copy code
#include <iostream>
int main() {
    std::cout << "Hello, World!";
    return 0;
}

C#

What it is: A language developed by Microsoft, designed for building Windows applications and games.
Uses: Game development (Unity), web development (ASP.NET), enterprise apps.
Example:
csharp
Copy code
using System;
class Program {
    static void Main() {
        Console.WriteLine("Hello, World!");
    }
}
Scripting Languages

PHP

What it is: A server-side scripting language for web development.
Uses: Dynamic websites, content management systems (e.g., WordPress).
Example:
php
Copy code
<?php
echo "Hello, World!";
?>

Ruby

What it is: Known for its elegance and simplicity, often paired with the Ruby on Rails framework.
Uses: Web applications, scripting.
Example:
ruby
Copy code
puts "Hello, World!"

Bash

What it is: A command-line scripting language used in Unix/Linux systems.
Uses: Automation of tasks, system management.
Example:
bash
Copy code
echo "Hello, World!"
Data Science & Statistical Computing

R

What it is: A language focused on statistical analysis and data visualization.
Uses: Data science, machine learning, bioinformatics.
Example:
r
Copy code
print("Hello, World!")

MATLAB

What it is: A language used for numerical computing and simulations.
Uses: Engineering, physics, data analysis.
Example:
matlab
Copy code
disp('Hello, World!')
Web Development Frameworks and Markup

HTML

What it is: The standard language for structuring web pages.
Uses: Building the skeleton of websites.
Example:
html
Copy code
<h1>Hello, World!</h1>

CSS

What it is: Used for styling HTML content.
Uses: Designing visually appealing web pages.
Example:
css
Copy code
h1 {
    color: blue;
}
Specialized Languages

SQL

What it is: A language for managing and querying relational databases.
Uses: Data retrieval, database management.
Example:
sql
Copy code
SELECT "Hello, World!";

Swift

What it is: A language designed for Apple's ecosystem.
Uses: iOS and macOS app development.
Example:
swift
Copy code
print("Hello, World!")

Kotlin

What it is: A modern, concise language interoperable with Java.
Uses: Android development, backend services.
Example:
kotlin
Copy code
fun main() {
    println("Hello, World!")
}

Go (Golang)

What it is: A statically typed language developed by Google for performance and simplicity.
Uses: Web servers, cloud services, distributed systems.
Example:
go
Copy code
package main
import "fmt"
func main() {
    fmt.Println("Hello, World!")
}
Emerging and Miscellaneous

Rust

What it is: Focused on performance and safety, especially memory safety.
Uses: Systems programming, game development.
Example:
rust
Copy code
fn main() {
    println!("Hello, World!");
}

Dart

What it is: The language behind the Flutter framework.
Uses: Cross-platform mobile development.
Example:
dart
Copy code
void main() {
    print('Hello, World!');
}

TypeScript

What it is: A superset of JavaScript with static typing.
Uses: Large-scale web applications.
Example:
typescript
Copy code
console.log("Hello, World!");
